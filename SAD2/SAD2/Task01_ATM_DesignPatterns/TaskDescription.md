## Task Title
ATM System using Design Patterns and SOLID Principles

## Applied Concepts
- Singleton Pattern
- State Pattern
- SOLID Principles:
  - Single Responsibility Principle (SRP)
  - Open / Closed Principle (OCP)
- Unit Testing

## Task Description
This task implements an ATM system applying advanced design patterns and SOLID principles.
The system ensures a single ATM instance using the Singleton pattern and manages ATM states
using the State pattern. The design follows clean architecture principles and includes unit testing.
